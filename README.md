# Web Android Emulator

이 프로젝트는 브라우저에서 작동하는 클라우드 안드로이드 에뮬레이터입니다.  
다음 기능을 제공합니다:

✅ 가상 안드로이드 기기에서 앱 설치  
✅ 파일 업로드/다운로드  
✅ 구글 플레이 시뮬레이터  

## 실행 방법

### 1️⃣ 백엔드 실행
```bash
cd backend
npm install
node server.js
```

### 2️⃣ 프론트엔드 실행
```bash
cd frontend
npm install
npm run dev
```

브라우저에서 http://localhost:5173 접속
